# 排版
ButUI对文档排版重新设计,包括全局设置、标题、正文文本、列表等.

## 标题
所有 HTML 标题都可以通过 。&lt;h1&gt;&lt;h6&gt;

```html
<h1>h1. ButUI heading</h1>
<h2>h2. ButUI heading</h2>
<h3>h3. ButUI heading</h3>
<h4>h4. ButUI heading</h4>
<h5>h5. ButUI heading</h5>
<h6>h6. ButUI heading</h6>
```
### 副标题
可以通过&lt;small&gt;元素来定义副标题

```html
<h1>Fancy display heading<small>With faded secondary text</small></h1>
```

## 显示标题
在某些用于宣传的特殊场合,可能需要使用更大的标题来配合页面设计.
