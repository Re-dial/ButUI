# 菜单
集成多个选项的菜单组件

基础用法
```html
<div>
	<but-menu>
		<but-button>菜单</but-button>
		<but-menu-content>
			<but-menu-item>项目 1</but-menu-item>
			<but-menu-item>项目 2</but-menu-item>
			<but-menu-item>项目 3</but-menu-item>
		</but-menu-content>
	</but-menu>
</div>
```
