# but-ui
这是一个基于 Vue 2.0 的桌面端组件库

## 项目设置
```
yarn install
```

### 编译热装载环境以进行开发
```
yarn run serve
```

### 编译压缩后的说明文档
```
yarn run build
```

### 编译压缩后的生产版本
```
yarn run lib
```

### 运行测试
```
yarn run test
```

### 整理和修复文件
```
yarn run lint
```
